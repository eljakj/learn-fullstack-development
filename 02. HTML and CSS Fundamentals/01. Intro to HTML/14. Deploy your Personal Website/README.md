# Deploy deine persönliche Webseite

1. Öffne **netlify.com**.
2. Klicke auf den **Sign up**-Button.
3. Wähle **Sign up with email**.
4. Gib deine **E-Mail-Adresse** und dein **Passwort** in die Eingabefelder ein.
5. Du erhältst eine **Bestätigungs-E-Mail**, die du bestätigen musst.
6. Beantworte die Fragen während der Registrierung.
7. Klicke im markierten Bereich auf **Browse project** und lade dein Projekt hoch.
